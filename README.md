## How to start
- Link: https://reactnative.dev/docs/environment-setup
- Link: https://docs.expo.dev/guides/typescript/

## How create Expo Go project
- npx create-expo-app -t expo-template-blank-typescript
- Name project
- Download Expo Go application

## What to check
- Node.js version:
	- Command: node -v
	- Recommended version: 18.14.1
- npm/npx version:
	- Command: npm --v
	- Recommended version: 9.5.0


## Basic commands
- Run project:
	- npm start
- Clean and check code:
	- npm run clean-code
